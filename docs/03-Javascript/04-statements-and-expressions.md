# Statements and Expressions

Statements 指一行一行由電腦執行的程式碼，而 Expressions 指 statements 裡會產生 value 的程式碼。

```js
//statements

if (a > b) {
    console.log(a)
} else {
    console.log(b)
}

//Expressions
a
b
a > b ? console.log(a) : console.log(b)
```
